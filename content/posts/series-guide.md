# 合集功能使用指南

合集功能已经成功添加到你的博客中！以下是使用方法：

## 在文章中添加合集

在文章的 Front Matter 中添加 `series` 字段：

```toml
+++
title = '文章标题'
date = 2026-03-15T10:00:00+08:00
draft = false
categories = ["分类"]
tags = ["标签1", "标签2"]
series = ["合集名称"]
+++
```

或者使用 YAML 格式：

```yaml
---
title: "文章标题"
date: 2026-03-15T10:00:00+08:00
draft: false
categories: ["分类"]
tags: ["标签1", "标签2"]
series: ["合集名称"]
---
```

## 功能特性

1. **导航栏入口**：点击顶部导航栏的"合集"可以查看所有合集列表
2. **文章页面显示**：在文章底部会显示该文章所属的合集
3. **合集页面**：点击合集名称可以查看该合集下的所有文章
4. **多合集支持**：一篇文章可以属于多个合集
